# EnStack Landing

Marketing site for EnStack. Built with Next.js, Tailwind CSS, and TypeScript.

Live at [enstack.co.uk](https://enstack.co.uk).

## Setup

```bash
pnpm install
pnpm dev
```

## Deployment

Static export deployed to Cloudflare Pages. Domain registered via GoDaddy.

```bash
pnpm build
```

Output is in the `out/` directory.
